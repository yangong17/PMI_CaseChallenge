4. Control Overview

Overview

The Control process group focuses on monitoring project performance and ensuring that progress remains aligned with the project plan. It involves measuring actual outcomes against the baselines for scope, schedule, cost, and quality, and taking corrective actions when deviations occur.

In this phase, the team is responsible for maintaining visibility into the project's performance and ensuring that all approved changes from the execution phase are properly integrated into the project documentation and reporting tools.

During this phase, the team should:

  • Update the cost baseline diagram to reflect budget changes from approved requests
  • Verify that schedule and risk updates are consistent with revised project scope
  • Document major variances and explain any corrective actions taken

Overall, Control is about managing change deliberately to maintain alignment with project goals.

Control Elements

  • Performance Measurement
  • Variance Analysis
  • Trend Analysis
  • Corrective Actions
  • Baseline Management
