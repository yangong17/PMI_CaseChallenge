Case Challenge and Final Presentation Guidelines

Challenge Structure

This case challenge is made up of 10 modules, each presenting a progressive scenario you might encounter in a real project.

By the end, you will bring together everything youâ€™ve developed into a final presentation that tells the full story of your project journey.

Module Components

Each module will have some of the following components:

  • Overviews of the scenario, outlining the objective(s)
  • Memos from key stakeholders to guide your decision making
  • Templates for you to craft your deliverables

Challenge Structure

The case challenge is organized into five main phases, each corresponding to a project management process group:

Phase Description
Project Initiation Develop project charters and stakeholder registers that align with organizational objectives
Project Planning Create comprehensive project plans including work breakdown structures, schedules, cost baselines, and risk management strategies.
Project Execution Adapt to changing circumstances. Manage project execution while handling changes and stakeholder expectations.
Project Control Monitor project performance and implement corrective actions as needed. Develop data visualizations to communicate project performance.
Project Closure Conduct project closure activities and document lessons learned. Develop a final presentation summarizing your project journey.

Final Presentation Rubric

Your final presentation should include the following sections:

Section Description
1. Executive Summary Overview of the project
2. Problem Statement and Objectives Why was the project initiated? What were your objectives?
3. Scope and Key Planning Elements How was the project broken down? (Scope, WBS, and major assumptions)
4. Baseline Performance Measures What was the planned schedule and cost baselines?
5. Risk and Uncertainty Analysis What were the associated risks and cost uncertainty outputs?
6. Change Management Record Were there any unexpected events? How did they affect the project? Did you have to update any projections?
7. Final Performance Report How did your results compare with your projections? How did you perform overall?
8. Lessons Learned What were your key takeaways from this challenge? Is there anything you would have done differently?
