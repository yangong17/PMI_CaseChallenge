2.2. Developing the Schedule

Scenario Continuation

You've now reached the next phase: developing the project schedule. To do this effectively, you'll need to gather the following information:

  • The required tasks and key milestones
  • The duration of each task
  • The dependencies between tasks

Since the Plant Manager has previously managed the installation of a paint booth, much of this information is already available. He has provided a detailed breakdown of steps, durations, and dependencies in a memo.

Your next task is to take that information and structure it into a Gantt Chart.

Objective

Create a Gantt Chart that:

  • Shows all the required steps, with task durations, dependencies, and milestones
  • Shows the critical path
Tip: Gantt Charts can easily be made on Smartsheet.com, which you can access on a 30-day free trial. You will also need to activate dependencies in Smartsheet to accurately build the Gantt Chart for this project.
